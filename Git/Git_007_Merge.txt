
當有多個 branch 時可以做 merge 來合併內容

---

合併時，先 checkout 到主枝幹

然後輸入

$ git merge [branch_name（副）]

來將副枝幹合併到主枝幹

---

Fast forward :

當主枝幹正好在副枝幹前一個

Git 會將主枝幹直接跳到副枝幹

---

兩個分支：

Git 會創造一個新的 snapshot 為這兩個分支的 child

讓他們在此點合併

若有衝突需處理

---

Conflict :

Git 會暫停 merge 的程序

你可以用 git status 來查看那些檔案有衝突

此時進去那個檔案看，可以看到以下情況

<<<<<<<
你所在的 branch 的內容
=======
副枝幹 branch 的內容
>>>>>>>

此時可以直接修改此部分，最後要把<<<<<<<、========、>>>>>>>>都拿掉

如此一來最後的結果就是你修改後保留的結果

最後再 git commit 把 merge 程序完成，新的一個 commit 就誕生了
