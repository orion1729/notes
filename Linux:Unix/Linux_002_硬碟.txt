IDE 硬碟在 Linux 的檔名是 hd[a - d]

一般主機會有兩個 IDE 插槽，每個可以裝兩個 IDE 裝置（Master, Slave）
IDE1 的插槽分別為 hda（Master）、hdb（Slave）
IDE2 則是 hdc、hdd

---

SATA 硬碟則是依據 Linux 捕捉到的順序
依照順序為 hd[a - p]

---

硬碟在切割時，最小單位是磁柱

---

硬碟的第一個磁區（512 bytes）
是給系統放開機管理程式（Master Boot Record, 446 bytes）、磁碟分割表（Partition table, 64 bytes）

磁碟分割表只能儲存四個切割的資訊

若要切割更多，需要在第二個切割中存入一個表
然後在第二個切割槽中來做剩餘的切割（Logical Partition）

檔名 hda[1 - 4] 保留給了磁碟分割表的四個
若使用邏輯分割，則會從 5 開始

IDE 最多可分割 59 個（5 至 63）
SATA 最多 11 個（5 至 15）

---

開機程序：
1. 硬體系統使用內嵌的 BIOS
2. BIOS 開始執行，找到一個可開機磁碟的開機磁區（第一磁區）內的開機管理程式
3. 開機管理程式可以讀取核心檔案或者呼叫其他開機管理程式
4. 核心開始執行，即作業系統的功能

多重開機：
由於每個分割槽都可以擁有自己的開機磁區
所以我們把作業各別灌在不同的分割槽（把核心塞入不同分割槽）
配合開機程序的敘述，我們可以連結多個開機管理程式達成多重開機

多重作業系統：
先安裝 Windows 再裝 Linux
因為 Windows 會覆蓋 MBR 與其安裝的分割槽所擁有的開機磁區
而 Linux 在安裝時可以選擇將開機管理程式所存放的位置
並且可以設定開機選單（即可以增加 Windows 選項）
