在 HTML 中，瀏覽器的顯示分為兩種排列方式

- Block level
  遇到此種排列方式，瀏覽器會從新的一行開始，並在新的一行結束
  像 <h1>, <p>, <ul>, <table> 都屬於此類

- Inline level
  inline 則是不換行
  <b>, <td>, <a>, <img> 則屬於此類

---

<div>, <span> 標籤：

兩個標籤都是用來當作容器用的

目的是改變原本利用 <table> 來作網頁排版的不好習慣

<div> 標籤是 block level，<span> 則是 inline

兩者的差別僅於此，兩者都是於排版時作為其他標籤的容器來使用

可用 CSS 來對這兩個標籤作調整



<div> 可用 CSS 的 float 來掩蓋 block 效果
